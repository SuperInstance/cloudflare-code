## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/claudeflare/claudeflare.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Deploy to production**
   ```bash
   npm run deploy
   ```

That's it. Your AI-powered application is now live globally.
