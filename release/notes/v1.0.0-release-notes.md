## Breaking Changes

### Migration from v0.x to v1.0

1. **API Endpoint Changes**
   - `/api/v1/` prefix required for all endpoints
   - WebSocket endpoint changed from `/ws` to `/api/v1/ws`

2. **Authentication**
   - JWT tokens now required for all API calls
   - API keys deprecated in favor of OAuth 2.0

3. **Configuration**
   - Environment variables renamed (see Migration Guide)
   - `wrangler.toml` structure updated

4. **Storage**
   - R2 bucket name format changed
   - D1 database schema migrated to v1.0

**See [Migration Guide](./migration-guide.md) for detailed instructions.**

---

## Deprecated Features

The following features are deprecated and will be removed in v1.1:

- API key authentication (use OAuth 2.0)
- Direct D1 access (use API endpoints)
- Legacy WebSocket protocol (use v1.1 protocol)
- Python 3.8 support (requires Python 3.9+)

## Known Issues

1. **Large File Uploads**
   - Files >25MB may timeout on free tier Workers
   - **Workaround:** Use R2 presigned URLs

2. **WebSocket Reconnection**
   - Automatic reconnection may fail after network changes
   - **Workaround:** Implement manual reconnection logic

3. **Concurrent Agent Sessions**
   - >1000 concurrent sessions may degrade performance
   - **Fix:** Planned for v1.0.1

4. **Cold Start Latency**
   - First request may take 2-3 seconds
   - **Mitigation:** Use warming pings

## Upgrade Instructions

### From v0.9.x to v1.0.0

```bash
# Backup current deployment
npm run backup

# Update dependencies
npm update

# Run database migrations
npm run db:migrate

# Update configuration
cp .env.v1.example .env

# Deploy to staging first
npm run deploy:staging

# Run smoke tests
npm run test:smoke

# Deploy to production
npm run deploy:production

# Verify deployment
npm run verify:production
```

### First-Time Installation

```bash
# Clone repository
git clone https://github.com/your-org/claudeflare.git
cd claudeflare

# Install dependencies
npm install

# Configure environment
cp .env.example .env
nano .env

# Initialize database
npm run db:init

# Deploy to development
npm run deploy:dev

# Run tests
npm test
```
